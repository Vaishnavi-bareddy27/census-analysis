Census Data Analysis

This repository contains an analysis of census data using Python and Matplotlib. The project reads a CSV dataset from Kaggle, explores the data, and creates visualizations to understand population distribution, demographics, and other key insights.

🗂️ Dataset

Source: Kaggle Census Dataset

File: census.csv

Description: The dataset contains census information such as population, gender distribution, and other demographic features.


📊 Visualizations

The project uses Matplotlib to generate various plots:

Histogram: To show distribution of numerical data

Bar Plot: To compare categorical data like population by region

Pie Chart: To show proportions of different groups

Line Graphs: To track trends over years or categories


🛠️ Technologies Used

Python 3.x – For data manipulation and visualization

Pandas – To read and process CSV data

Matplotlib – To create plots and visualizations
